### 1.2 SampleApp 集成方式重构

**旧流程**：
```
build → file(COPY) 从 QAIRT_SDK/examples/.../SampleApp 到 3rdparty/SampleApp
      → git apply SampleApp.patch  （把 mmap、convertToFloatInto、private→public 等魔改合入）
```

**新流程**：
- 直接把 **2.45 SDK 的 SampleApp 源码预拷贝并 check-in** 到 `app/src/main/cpp/3rdparty/SampleApp/`
- 手工把原 `SampleApp.patch` 的所有修改（mmap 读 cached binary、`convertToFloatInto` 新增接口、`private:` 注释成可继承）合入到 4 个文件：
  - `QnnSampleApp.cpp`
  - `QnnSampleApp.hpp`
  - `Utils/IOTensor.cpp`
  - `Utils/IOTensor.hpp`
- `SampleApp.patch` 文件保留在仓库根（193 行）作为历史参考，但**不再由构建调用**
- 验证：`grep -n "mmap\|convertToFloatInto" 3rdparty/SampleApp/src/...` → 已确认 4 处修改全部在场
